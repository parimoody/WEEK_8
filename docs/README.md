## Overview
This project implements a SQLite database backend for the Multi-Domain
Intelligence Platform. It replaces file-based storage with a relational
database and supports secure authentication and CRUD operations.

---

## Features
- SQLite database using Python sqlite3
- Secure user authentication with bcrypt
- CRUD operations for cyber incidents
- SQL injection protection using parameterized queries

---

## Database Tables
- users
- cyber_incidents
- datasets_metadata
- it_tickets
